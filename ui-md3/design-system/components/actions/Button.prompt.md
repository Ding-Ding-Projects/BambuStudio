Pill button in the five variants Bambu Studio MD3 uses; filled = the one primary action per surface.

```jsx
<Button variant="filled" size="lg" icon="print" iconFill elevated>Print plate</Button>
<Button variant="outlined" size="lg" icon="deployed_code">Slice plate</Button>
<Button variant="text">Cancel</Button>
```

Danger variant is outlined error (Stop). Font 13.5px/600 (500 outlined); radius = height/2.
